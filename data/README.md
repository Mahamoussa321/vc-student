# Data directory

This folder is reserved for input data needed to reproduce the analysis.

Large raw data files are not tracked in GitHub.

Expected local files include:

- `point_data_with_atmospheric_predictors.csv`: station-level rain--snow modeling data with atmospheric predictors.
- `Koeppen-Geiger-ASCII.txt`: Köppen--Geiger climate-zone grid used for climate-boundary map backgrounds.

Because some files are large, users should place them in this folder before running the scripts.
